Electric Vehicle Charging Networks (GET /api/alt-fuel-stations/v1/electric-networks)

Return a full list of electric vehicle (EV) charging networks. This dataset powers the Electric Vehicle Charging Networks on the Alternative Fuels Data Center (AFDC).

Request URL

GET /api/alt-fuel-stations/v1/electric-networks.json?parameters

Request Parameters

The following query parameters may be passed in the request to control the output:

Parameter Required Value Description
api_key Yes
Type: string
Default: None

Your developer API key. See API key usage for more information.
country No
Type: string
Default: US
Options: all, US, CA

Return only networks within the given country code.

Value Description
all U.S. and Canada
US United States
CA Canada
include_inactive No
Type: boolean
Default: false
Options: true, false

If true, then include inactive EV charging networks that are not visible on the AFDC.

Response Fields

The response is composed of a list of electric vehicle charging networks matching the requested query. The network data is provided in the following structure.

Field Value Description
key
Type: string

The key that identifies the charging network.
evchart_key
Type: string

A key in use for EV-ChART.
name
Type: string

EV charging network name.
url
Type: string

The URL for the charging network.
last_import_date
Type: date

The most recent date for which equipment data has been confirmed.
date_added
Type: date

The date the network was added to the AFDC.
date_removed
Type: date

The date the network was removed from the AFDC.
import_type
Type: string

The method by which data about the network is imported.

Value Description
CSV CSV import
API API import
Manual Data is manually entered
transition_network_key
Type: string

For networks removed from the AFDC, the key of the acquiring network, if any.
transition_note
Type: string

For networks that acquired or merged with other networks, a comma-separated list of the former network names.
transition_timeframe
Type: string

For networks removed from the AFDC, the timeframe when the network acquisition or transition was complete.
