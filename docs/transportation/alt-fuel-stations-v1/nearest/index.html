Nearest Stations (GET /api/alt-fuel-stations/v1/nearest)

Return the nearest alternative fuel stations within a distance of a given location. This dataset powers the Alternative Fueling Station Locator on the Alternative Fuels Data Center.

Request URL

GET /api/alt-fuel-stations/v1/nearest.format?parameters

Request Parameters

The following query parameters may be passed in the request to control the output:

Parameter Required Value Description
format Yes
Type: string
Default: None
Options: json, xml, csv, geojson, kml

The output response format. Given in the URL as the file extension.
api_key Yes
Type: string
Default: None

Your developer API key. See API key usage for more information.
response_format No
Type: string
Default: None
Options: compact

Return a simplified response for fuel station records with only these fields included: id, fuel_type_code, latitude, and longitude.
latitude Yes
Type: decimal
Range: -90 to 90

The latitude of the desired location.
longitude Yes
Type: decimal
Range: -180 to 180

The longitude of the desired location.
radius No
Type: decimal or string
Default: 5.0
Range: 0.0 to 500.0, or infinite

The radius (in miles) around the search location to search for stations within.

An explicit radius of up to 500.0 miles may be passed in, or the special infinite string may be passed in to find the nearest stations regardless of distance.

ev_connector_type_operator No
Type: string
Default: OR
Options: OR, AND

Control how multiple connector type options passed to the ev_connector_type parameter behave. The default of OR will return stations that have any of the connectors present. Specifying AND will only return stations that have all of the connectors present.

For example, with the default OR value, then performing a query for ev_connector_type=CHADEMO,J1772COMBO will return any station that has either CHADEMO or J1772COMBO connectors present. Compare that to a query for ev_connector_type=CHADEMO,J1772COMBO​&ev_connector_type_operator=AND which will return stations that must have both CHADEMO and J1772COMBO connectors present at the same station.

Response Fields

The response is composed of the fuel stations matching the requested query. For JSON, GeoJSON, or XML formats, additional metadata is supplied in the following structure:

Field Value Description
total_results
Type: integer
How many total fuel station records match your requested query (regardless of any limit applied).
station_counts
Type: record
A record that details the total number of stations and charging outlets that match your requested query (regardless of any limit applied). See station count record fields for response details.
station_locator_url
Type: string
The URL of the Alternative Fueling Station Locator, the web application that supplies this data.
offset
Type: integer

The result offset applied to this query (if the "offset" parameter was given).
latitude Type: decimal The latitude of the given input location.
longitude Type: decimal The longitude of the given input location.
fuel_stations
Type: array
An array of matching fuel station records returned in order of ascending distance from the given location. See fuel station record fields for response details.
