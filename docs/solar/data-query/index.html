Solar Dataset Query

Returns information about data available for a given location for the solar resource database used by the PVWatts® Calculator and PVWatts APIs.

Solar Dataset Query V1 (GET /api/solar/data_query/v1)

Returns information about data available for a given location for the solar resource database used by the PVWatts v6 API.

Solar Dataset Query V2 (GET /api/solar/data_query/v2)

Returns information about data available for a given location for the solar resource database used by the PVWatts® Calculator and PVWatts v8 API.
