Energy Optimization

Tools and models to optimize renewable energy, conventional generation, and energy storage systems for buildings, campuses, communities, and microgrids.

RE Data Explorer Technical Potential

Technical Potential analysis tool for The Renewable Energy Explorer which estimates the technical potential of specific renewable electricity generation technologies taking into account a variety of user supplied filters and other inputs.

REopt Lite™ API (Version 1)

The REopt™ API provides access to a web version of a techno-economic decision support model used to optimize energy systems for buildings, campuses, communities, and microgrids.
